# E-Shop 电商平台

## 项目简介
基于 Vue 3 + Vant 4 的移动端电商平台，包含完整的购物流程和用户系统。

## 功能特性
- 🏠 首页：轮播图、分类导航、商品展示
- 🗂️ 分类：商品分类浏览、搜索
- 🛍️ 购物车：商品管理、数量调整、批量操作
- 👤 个人中心：用户信息、订单管理
- 📱 响应式设计：完美适配移动端
- 💾 数据持久化：购物车数据本地存储

## 技术栈
- Vue 3 (Composition API)
- Vue Router 4 (路由管理)
- Pinia (状态管理)
- Vite (构建工具)
- Vant 4 (UI组件库)
- Axios (HTTP请求)
